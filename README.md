# AntarX: Juices & Desserts Platform

Welcome to the AntarX platform, a premium digital product built for scale and high-end retail/B2B/Event experiences.

## 🚀 Project Overview
This repository contains the architecture and initial codebase for the Juices & Desserts shop, featuring a customer-facing website and a private administrative dashboard.

### Key Features
- **Bilingual (AR/EN)**: Full RTL support.
- **Dual-Auth**: Completely isolated databases for Customers and Admins.
- **Theme-Aware**: Seamless light/dark mode transitions.
- **Three-Tier Clients**: Individuals, Businesses, and Event Creators.

---

## 📂 Project Structure
- `/customer-web`: Next.js frontend for the public store.
- `/admin-dashboard`: Next.js dashboard for internal management.

---

## 🛠 Tech Stack
- **Frontend**: Next.js 14+
- **Styling**: Vanilla CSS / CSS Modules (Premium Aesthetics)
- **Database**: Supabase / PostgreSQL
- **Auth**: NextAuth / Supabase Auth
